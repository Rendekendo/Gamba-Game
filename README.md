# Gamba Game

A simple web-based game built with React where you can earn money, gamble, (deposit to a bank or withdraw TBA)

# How to run

- [Node.js](https://nodejs.org/) installed
- `cd Gamba-Game`
- `npm install`
- `npm run dev`
